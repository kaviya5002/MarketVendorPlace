HOW TO RUN FROM INTELLIJ IDEA
========================================

Since Maven is not available in your command line, here's how to run from IntelliJ:

METHOD 1: Run Main Application Class
-------------------------------------
1. Open IntelliJ IDEA
2. Open the project: CampusEcommerceWeb
3. Navigate to: src/main/java/com/campus/ecommerce/CampusEcommerceWebApplication.java
4. Right-click on the file
5. Select "Run 'CampusEcommerceWebApplication.main()'"
6. Wait for the server to start (you'll see "Started CampusEcommerceWebApplication" in console)
7. Open browser and go to: http://localhost:8080/index.html

METHOD 2: Use Maven Tool Window
--------------------------------
1. Open IntelliJ IDEA
2. Click on "Maven" tab on the right side
3. Expand "CampusEcommerceWeb" ‚Üí "Plugins" ‚Üí "spring-boot"
4. Double-click on "spring-boot:run"
5. Wait for the server to start
6. Open browser and go to: http://localhost:8080/index.html

METHOD 3: Use Run Configuration
--------------------------------
1. Open IntelliJ IDEA
2. Click "Run" menu ‚Üí "Edit Configurations"
3. Click "+" ‚Üí "Spring Boot"
4. Name: "Campus Ecommerce Server"
5. Main class: com.campus.ecommerce.CampusEcommerceWebApplication
6. Click "OK"
7. Click the green "Run" button
8. Open browser and go to: http://localhost:8080/index.html

TROUBLESHOOTING:
----------------
‚ùå If you get "Port 8080 already in use":
   - Stop any other application using port 8080
   - Or change the port in application.properties: server.port=8081

‚ùå If you get database connection errors:
   - Make sure MySQL is running
   - Check username/password in application.properties
   - Verify database 'campus_ecommerce' exists

‚ùå If IntelliJ shows Maven errors:
   - Click "Maven" tab ‚Üí Click refresh icon (circular arrows)
   - Or: File ‚Üí Invalidate Caches ‚Üí Restart
